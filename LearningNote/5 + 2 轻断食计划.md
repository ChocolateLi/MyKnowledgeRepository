# 5 + 2 轻断食计划

这是一份**5+2 全营养轻断食执行方案**。这份计划遵循“控卡不控营养”的原则，结合了16+8饮食法，旨在帮助你在控制热量的同时保证基础营养摄入。

---

# 📅 一、核心理念与基本规则

1.  **5+2 断食节奏**：
    *   **5天**：正常进食日（非断食日），保持营养均衡和低碳饮食。
    *   **2天**：轻断食日（如周二/周五，或周三/周六），严格控制热量。
2.  **16+8 进食窗口（轻断食日适用）**：
    *   将一天中所有的食物在 **8小时内** 吃完。
    *   *适应期建议*：初期可将时间放宽至8-10小时（例如：早8点至晚6点），待身体适应后再收紧至8小时。
3.  **热量控制**：
    *   轻断食日总热量控制在 **500千卡以内**。根据你的营养素计算，本方案约为 **411千卡**，留有弹性空间。

---

# 🗓️ 二、轻断食日（热量控制在 500千卡以内）

这一天的核心是**定时补充营养，大量饮水**。

*   **总热量计算**：
    *   每份组合：1勺蛋白质粉 (32千卡) + 1袋乐纤 (105千卡) = **137千卡**
    *   一日三餐合计：137千卡 × 3 = **411千卡**
    *   *（倍立健和B族维生素热量极低，主要补充微量元素，忽略不计）*

*   **执行时间表**：

    *   **🕖 07:00 早餐**
        *   营养素：蛋白质粉 1勺 + 乐纤 1袋 + 倍立健 1对 + B族维生素 3片
        *   用法：倍立健和B族直接温水吞服；蛋白质粉和乐纤用摇摇杯加300ml温开水摇匀饮用。

    *   **🕚 11:00 - 12:00 午餐**
        *   营养素：蛋白质粉 1勺 + 乐纤 1袋 + 倍立健 1对 + B族维生素 3片
        *   用法：同上。

    *   **🕒 15:00 - 17:00 晚餐**
        *   营养素：蛋白质粉 1勺 + 乐纤 1袋 + 倍立健 1对 + B族维生素 3片
        *   用法：同上。

*   **⚠️ 重要注意事项**：
    *   **禁食**：除以上三餐外，其他时间**不吃任何食物**（包括口香糖、含糖饮料等）。
    *   **饮水**：必须保证全天饮水量达到 **2000ml 以上**，建议饮用优质的矿物质水（温开水为宜）。这有助于提高代谢，缓解饥饿感。

---

# 🥗 三、非轻断食日（营养均衡日）

这5天不需要严格控卡，但要建立良好的饮食习惯，为下一次断食做准备。

## 1. 营养补充（早晚强化）
*   **早饭前半小时**：蛋白质粉 1勺 + 倍立健 1对 + B族维生素 3片
*   **晚饭前半小时**：蛋白质粉 1勺 + 倍立健 1对 + B族维生素 3片
*   **睡前规则**：晚上8点后不再进食，如口渴可少量喝水。

## 2. 低碳饮食执行细则
*   **进食顺序**（调整后有助于控制血糖和食欲）：
    1.  **先喝汤**（清淡的汤，非浓汤或勾芡汤）
    2.  **再吃菜**（蔬菜为主，特别是绿叶菜）
    3.  **再吃肉/蛋/豆**（蛋白质来源）
    4.  **最后吃主食**（米面薯类等）
*   **主食控制**：
    *   **原则**：有意识地逐渐减少主食量，每顿吃 **七分饱** 即可。
    *   **渐进法**：
        *   习惯期（第一周）：从平时的饭量（如1碗饭）减少至 3/4 碗。
        *   稳定期（后续）：根据身体感受，可进一步减至半碗或更少，甚至可以尝试在某些正餐完全不吃主食。
*   **常见主食清单**：
    *   **谷物类**：米饭、馒头、面条、粥、包子皮、饺子皮、米粉、肠粉、拉肠。
    *   **根茎类（碳水较高）**：马铃薯（土豆）、芋头、玉米、莲藕、南瓜、番薯（红薯）、山药。

# 📝 特别提醒

1.  **灵活性**：轻断食的两天不必固定，根据你的工作排班（如周二周五）灵活调整，只要保证两天**不连续**即可。
2.  **适应期反应**：刚开始断食日可能会有轻微饥饿感、头晕或情绪波动，请保证充足饮水，通常坚持2-3次后身体会逐渐适应。
3.  **身体反馈**：如果在非断食日感到极度疲惫或精力不济，可以适当增加优质脂肪（如坚果、牛油果）或蛋白质的摄入，不必过于严苛。

# 🔬 营养素的作用

## B族维生素

**核心作用**

**1.能量代谢（燃脂引擎）**：
B族维生素（特别是B1、B2、B3、B5、B7）是帮助身体将摄入的蛋白质、脂肪和碳水化合物转化为可用能量（ATP）的关键辅酶。

**2.神经系统稳定剂**

**维生素B1（滋养神经）、B6（调节情绪）、B12（修复神经鞘）**：有助于维持神经系统的正常功能。

**3.维护皮肤与粘膜健康**

**维生素B2、B3、B5**：对皮肤健康很重要。在断食和营养调整期间，有助于预防口角炎、唇炎等。

**4.红细胞生成**

**维生素B9（叶酸）和B12**：参与红细胞的合成，保证身体的氧气运输能力。
